Estilos externos

Entendimento:

Estilos externos permitem a criação de estilos em css para serem utilizados em
várias páginas simûntaneamentes. Este método garante uma melhor organização
do código html e uma melhor versatilidade para mudanças do mesmo design a ser
utilzado em mais de uma página, diminuindo repetições de pré-definições.
