Estilos locais/ internos

Entendimento

Estilos locais permitem uma pré-definição dos estilos das tags semânticas em um body { }". Este tipo melhora a organização do código html separando o controle de design/ formatação e os conteúdos a serem formatados. A desvantagem é gerar um código de formatação muito grande até chegar ao conteúdo, de fato. Outro problema é a necessidade de uma pré-formatação nas demais páginas, mesmo que seja o mesmo design.
